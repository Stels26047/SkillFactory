В данном файле продемонстрирован код в котором компьютер пытается угадать задуманное число пользователя в функции.
В итоге выводится результат и средние значение за 1000 попыток.

This file demonstrates the code in which the computer tries to guess the intended number of the user in the function.
As a result, the result and the average value for 1000 attempts are displayed.
